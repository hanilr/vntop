### v0.1.3 (Processes)

- Removable disks will be shown as `[Ext 'label']`.
- Empty disk name fix. If there is no label will be shown as 'No Label'.
- Keymap switched to function keys.
- `is_keymap` algorithm update.
- Process kill add.
- Process search add.

### v0.1.2 (Visuals)

- Release notes add. 
- Roadmap update.
- Process sorting and processes widget visual update.
- Keymap add.
- `goto` add to UI.
- `cargo test` add.
- Key and resize event optimization.
- `/img` update.

### v0.1.1 (Live Loop)

- Changed `terminal_size` to `crossterm` crate.

- Byte and time format add. Support Byte to Terabyte.
- Keybind basics.
- Terminal resize event add.
- Terminal screen clean add.
